# Presentation Materials

Everything you need to present this project to a class.

| File | What It Is |
|------|-----------|
| [SCRIPT.md](SCRIPT.md) | Full speaker script with exact words, commands, and timing |
| [../docs/presentation.pptx](../docs/presentation.pptx) | 15-slide deck (dark navy theme) |
| [../docs/05-run-demo.md](../docs/05-run-demo.md) | Standalone demo instructions (if you want just the demo) |
| [EXERCISES.md](EXERCISES.md) | Student exercises (local Docker Compose, no AWS needed) |

## Quick Prep

1. Read `SCRIPT.md` end to end once
2. Start Docker: `cd local && docker compose up --build -d`
3. Open slides, AKHQ (localhost:8080), and a terminal
4. Give the talk

## Timing

~40 minutes talk + 10 minutes Q&A. See the timing table in SCRIPT.md for where to cut if running long.
